Introduction to Processing

When you are finished this session you should be:

  • familiar with Processing and it's development environment (PDE).
  • able to use methods to draw basic shapes.
Formatting drawings

When you are finished this session you should be:

  • able to use methods to draw and format basic shapes.
  • familiar with Grayscale and RGB colours.
Animating your drawings

When you are finished this session you should be:

  • able to write code to animate simple drawings.
  • aware of the System Variables that come with Processing and be able to use them in your programs.
Data Types and Operators

When you are finished this session you should be:

  • comfortable with primitive data types and be able to use them in non-complex problems and calculations.
Conditional Statements

When you are finished this session you should be able to code animated drawings using the following constructs:

  • conditional statements
  • relational operators
  • logical operators
Event Handling

When you are finished this session you should be able to code animated drawings using the following constructs:

  • mouse events
  • key events
Iteration

When you are finished this session you should be:

  • able to write progams that engage in repetitive, looping behaviour.
  • familiar with variable scope and compound assignment statements.
