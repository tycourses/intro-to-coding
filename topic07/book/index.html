Objectives

On completion of this lab you should:

  • understand how to code static drawings using for loops.

Exercises

  • In these exercises, we will build start building an image piece by piece. You can use the same project for all these exercises.

Exercise 1 (static drawing)

  • Create a new sketch.
  • Using the code below, produce this drawing:

size(600, 300);
background(100);
noStroke();

int yCoordinate = 60;
//The for loop will run four times
for(int i = 0; i < 4; i++) {
  // Draw horizontal white bars 
  fill(255);
  rect(50, yCoordinate, 500, 10);
  yCoordinate+=20;
}
  • Can you see from the code how the image is drawn?

Exercises

  • In this exercise, we will tie together a lot of the concepts we have learned into one animated drawing.

  • We are going to create a "blackboard" and use a "duster" to rub off the contents of the blackboard:

    • the contents we will rub off will be similar to the series of rectangles you drew in the previous step.
    • the duster will be a circle that will follow the mouse pointer around.

Exercise 5 (animated drawing)

  • Create a new sketch.

  • Using the code below, produce this drawing:

  void setup() {
    size(500,400);
    background(0);

    //set up the blackboard with rectangles to "rub off"
    int yCoordinate = 20;
    for(int i = 0; i < 15; i++) 
    {
        // Draw horizontal white bars 
        fill(255);
        rect(50, yCoordinate, 350, 10);
        // Draw horizontal red bars 
        fill(255,0,0);
        rect(120, yCoordinate+20, 250 , 10);
        yCoordinate+=40;
     }

     //setting up the colour for the "duster"; we will draw it later
     fill(0,0,0);
  }

  • Can you see from the code how the image is drawn?

  • The next step is to add the draw method. This will draw our duster (a black circle) that will follow the mouse pointer. As the background is black, the circle will appear to "rub out" the white and red rectangles.

Add this code to your sketch:

  void draw() {

     //this is my "duster"
     ellipse(mouseX, mouseY, 50, 50);
  }

Run the code...does your "duster" rub out the white and red rectangles?

If so, how about adding something to the animation when we press the mouse button? Maybe we could clear the background and re-draw the white and red rectangles?

To do this, change your draw() method so that it looks like this:

  void draw() {

    if (mousePressed) {
       //redraw the background when the mouse is pressed
       background(0);
       //setup the blackborad with more contents
       int yCoordinate = 20;
       for(int i = 0; i < 15; i++) 
       {
          // Draw horizontal white bars 
          fill(255);
          rect(50, yCoordinate, 350, 10);
          // Draw horizontal red bars 
          fill(255,0,0);
          rect(120, yCoordinate+20, 250 , 10);
          yCoordinate+=40;
       }
       //setting up the colour for the "duster"
       fill(0,0,0);
    }

    //this is my "duster"
    ellipse(mouseX, mouseY, 50, 50);
  }

Notice that, in our draw() method we just repeated the "setting up the background" code that we wrote in the setup() method.

Run your code...does the background repaint itself when you click the mouse button?
