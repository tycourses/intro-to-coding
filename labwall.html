Lab-01

On completion of this lab you should:

  • be familiar with the PDE (Processing Development Environment).

  • be able to use functions to draw basic shapes.
Lab-02

On completion of this lab you should:

  • be more familiar with the PDE (Processing Development Environment).

  • be able to resize and colour your display window.

  • be able to add colour to basic shapes.
Lab-03

On completion of this lab you should:

  • be able to write code to animate simple drawings.
  • be aware of the system variables in the PDE and be able to use them effectively.
Lab-04

On completion of this lab you should:

  • understand primitive data types and be able to use a selection of them in non-complex problems and calculations.
Lab-05

On completion of this lab you should be able to code animated drawings using the following constructs:

  • conditional statements
  • relational operators
  • logical operators
Lab-06

On completion of this lab you should be able to code animated drawings using the following constructs:

  • mouse events
  • key events
Lab-07

On completion of this lab you should:

  • understand how to code static drawings using for loops.
Lab-08
  • This is the assignment for the coding course.
  • You are asked to write the code for a working animation.
