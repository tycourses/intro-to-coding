Objectives

On completion of this lab you should:

  • understand primitive data types and be able to use a selection of them in non-complex problems and calculations.

Primitive Data Types

  • In Java, there are:
    • four whole number data types: byte, short, int, long.
    • two decimal number data types: float and double.
    • a single character data type: char.
    • a true/false data type: boolean.
